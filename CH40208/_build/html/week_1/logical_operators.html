Logical operators¶

First, run the cell below and watch the video lecture associated with this topic.

Previously, we used the Jupyter Notebook as a simple calculator. However, programs are more useful when we are able to include some intelligence, allowing the code to perform different calculations under different circumstances. This ability relies heavily on the use of logical operators.

A logical operator is a piece of code that returns an object of type Boolean, which can be either True or False. The logical operator == is one of the most common and translates to is equal to, this operator will return True if the values on either side are equal, and False if they are not.

# The truth

print(2 - 1 == 1)
print(1 - 1 == 1)

In addition to the use of individual logical operators, it is possible to use additional keywords to combine multiple operators. These keywords are and and or, which has important but different actions:

  • The and keyword will return True if both operations are true.

  • The or keyword will return False if either operation is true.
